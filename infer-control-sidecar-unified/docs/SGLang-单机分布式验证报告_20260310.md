## 一、验证目标

在 150 单机上使用两张 RTX 4090，通过 StatefulSet (replicas=2) 部署两个 Pod，验证 wings-infer 的 **SGLang 分布式推理** 完整代码逻辑，包括：

1. Master/Worker 角色自动判断与协调
2. `distributed/master.py` — Master API 服务、Worker 注册、启动指令分发
3. `distributed/worker.py` — Worker API 服务、向 Master 注册、接收启动指令
4. `engines/sglang_adapter.py` — SGLang 分布式启动脚本生成 (`--nnodes`/`--node-rank`/`--dist-init-addr`)
5. PyTorch Distributed (NCCL/GLOO) 跨 Pod 通信
6. `proxy/` — 代理透传与健康检查
7. 端到端推理请求

## 四、关键设计与问题解决

### 4.1 复用 Pod 网络 + hostPath IP 交换方案

与 vLLM Ray 验证相同的基础设施方案，无需修改：
- **Pod 网络**: flannel CNI 为每个 Pod 分配独立 IP
- **IP 交换**: 通过 hostPath `/tmp/wings-ip-exchange` 交换 Pod IP
- **NODE_RANK**: 从 StatefulSet Pod 名称序号派生

### 4.2 nvidia-smi 不可用问题

**问题**: SGLang 在 `__post_init__` 中调用 `nvidia-smi` 获取 GPU 显存容量，k3s-in-Docker 环境中 SGLang 容器内 `nvidia-smi` 不在 `$PATH` 中。

```
OSError: [Errno 8] Exec format error: 'nvidia-smi'
```

**解决方案**: Engine 容器启动脚本中添加 PATH：
```bash
export PATH=/mnt/nvidia-libs:${PATH}
export LD_LIBRARY_PATH=/mnt/nvidia-libs:${LD_LIBRARY_PATH:-}
```

`nvidia-smi` 位于 hostPath `/mnt/nvidia-libs/nvidia-smi`（从宿主机驱动库中复制）。

### 4.3 SGLang 分布式参数流

代码调用链：

```
config_loader._merge_sglang_params()
├── _adjust_tensor_parallelism(tp_size, distributed=True)
│   └── tp_size = device_count(1) × n_nodes(2) = 2
├── ep_size = tp_size = 2     (sglang 4.10.0+ 替代 enable_ep_moe)
└── _handle_sglang_distributed()
    └── dist_port = 28030     (from distributed_config.json)

sglang_adapter.build_start_command()
└── python3 -m sglang.launch_server \
        --tp-size 2 --ep-size 2 \
        --nnodes 2 --node-rank {0|1} \
        --dist-init-addr {master_ip}:28030
```

### 4.4 NCCL 环境变量

| 变量 | 值 | 说明 |
|------|------|------|
| `NCCL_SOCKET_IFNAME` | eth0 | Pod 网络 CNI 接口 |
| `GLOO_SOCKET_IFNAME` | eth0 | GLOO 通信接口 |
| `NCCL_IB_DISABLE` | 1 | 禁用 InfiniBand |
| `NCCL_NET` | Socket | 使用 Socket 传输 |
| `NCCL_P2P_DISABLE` | 1 | 不同 Pod 无法 P2P |
| `NCCL_SHM_DISABLE` | 1 | 不同 Pod 无法共享 /dev/shm |

同时，`sglang_adapter.build_start_script()` 在脚本头部自动注入：
```bash
export GLOO_SOCKET_IFNAME=eth0
export TP_SOCKET_IFNAME=eth0
export NCCL_SOCKET_IFNAME=eth0
```
